## Как просматривать

Диаграммы написаны в формате **Mermaid**. Их можно открыть:

1. **GitHub** — Mermaid рендерится автоматически в `.md` файлах
2. **VS Code** — расширение «Markdown Preview Mermaid Support»
3. **Онлайн** — [mermaid.live](https://mermaid.live) — вставьте код из блока ` ```mermaid `
4. **Экспорт в PNG/SVG** — на mermaid.live: Share → Export

## Экспорт для ВКР

1. Откройте [mermaid.live](https://mermaid.live)
2. Скопируйте код диаграммы из соответствующего `.md` файла (блок между ` ```mermaid ` и ` ``` `)
3. Вставьте в редактор
4. Нажмите **Export** → PNG или SVG
5. Вставьте изображение в документ ВКР
